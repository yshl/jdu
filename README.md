# jdu
ディスクの使用量を表示するコマンドです。Windows用。

Windows Sysinternals の du が日本語ファイル名に対応していないようなので作ってみました。

Sysinternals の du との互換性はありません。

## 使用法
使用法: du [-l depth] [-c|-t] [-j] <ディレクトリ>

| オプション | 説明 |
|------------|------|
| -l depth | 結果出力するディレクトリの深さ |
| -c | 結果をコンマ区切りで出力 |
| -t | 結果をタブ区切りで出力 |
| -j | ジャンクションを含める |
| -h | ヘルプを表示 |

## ジャンクションについて
デフォルトではジャンクションはカウントしませんが、-j オプションを付けるとジャンクションも含めます。
